# Login Demo with [MeteorJS](https://www.meteor.com/), [Ethereum](https://ethereum.org/) & [MetaMask](https://metamask.io/)

## Installation
```sh
# select folder
$ cd meteorjs-ethereum-metamask-login-demo
# start app
$ meteor
```
